1. Что такое и зачем нужен HTML?

Это язык гипертекстовой разметки

2. Что такое и зачем нужен Git?

Git это система которая вклячает возможность создавать репозитарии для совмесной работой над проектами

3. Что такое SEO?

Это действие которое направлено на повышение отображения нужно сайта в топе поисковиков

4. В чем отличие Git от GitHub?

Git это программа установленная на локальной машине которая позволяет синхронизировать репозитории которые находяться на сервере GitHub

5. Что такое тег? Что такое атрибут?

Тег это команда которая используеться в HTML. Атрибут это дополнение к тегу которое расширяет его функционал или есть необходимым параметром для правильного отображения тега

6. Чем отличается язык разметки от языка програмирования? HTML можно назвать язіком программирования? Аргументируйте свою позицию.

Язык програмирования сам по сибе самодостаточен и с помощью него можно написать программы, а Html язые разметки который читаеться программой (браузером). Визуально очень похожи, но так как разметка Html всего лиш код который читает браузер с моей точки зрения ето не самодостаточный язык програмирования

7. Зачем нужен атрибут alt у изображений?

Атрибут alt у тега img нужен для отображения альтернативного текста картинки
